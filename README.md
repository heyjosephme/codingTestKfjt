# コーディングテスト課題集

このリポジトリには、以下の2つのTypeScriptプロジェクトが含まれています：

## プロジェクト一覧

### 1. factorial（階乗計算）
再帰関数を使用して正の整数Nの階乗を計算するプログラム。

- 詳細は[factorial/README.md](./factorial/README.md)を参照
- 機能:
  - 正の整数Nを入力として受け取り、N!を計算
  - TypeScriptの型定義
  - 再帰関数による実装
  - テストケース完備

### 2. trafficLight（信号機）
信号機のロジックを実装したプログラム。

- 詳細は[trafficLight/README.md](./trafficLight/README.md)を参照
